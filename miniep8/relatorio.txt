O problema escolhido foi retirado do site https://dojopuzzles.com/ e 
se trata de avaliar se um ano dado é bissexto ou não.
O link do programa se encontra em: https://dojopuzzles.com/problems/ano-bissexto/

Um ano é bissexto se for divisível por 4, mas não divisível por 100, 
exceto se ele for também divisível por 400.

Os testes nesse mini EP foram realizados com a lib pytest.

Nesse diretório no git, cada commit será pensado como um "ciclo" 
de TDD.

Conclusão sobre TDD:

O TDD parece um framework muito interessante para direcionar o 
desenvolvimento de software para ganhar agilidade e reduzir incidência
de bugs, contudo, talvez o problema aqui 
escolhido seja muito simples para ilustrar o uso correto.
